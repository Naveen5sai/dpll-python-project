# Digital Phase-Locked Loop (DPLL) using Python

## Overview
This project implements a Digital Phase-Locked Loop (DPLL) to synchronize the phase and frequency of an output signal with an input signal.

## Components
- Phase Detector
- Loop Filter (PI Controller)
- Numerically Controlled Oscillator (NCO)

## Tools Used
- Python
- NumPy
- Matplotlib

## Results
The PLL successfully locks onto the input signal, reducing phase error over time.

## How to Run
```bash
cd code
python dpll.py
